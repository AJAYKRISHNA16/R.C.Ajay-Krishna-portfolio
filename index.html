Research Projects

GNSS-INS integration in the embedded platform for reliable navigation

Developed embedded sensor fusion algorithms for GNSS–INS integrated navigation systems and implemented both tightly and loosely coupled integration frameworks

Developed an efficient loosely coupled GNSS/INS navigation algorithm using a nested UKF architecture for IMU calibration, integrated within a dual-layer estimation framework employing an inter-layer feedback mechanism based on MDOR and IAE to enhance estimation robustness and accuracy. Incorporated time synchronization, lever-arm compensation, and initial alignment within a tightly coupled framework to further improve system performance. Implemented real-time sensor fusion and trajectory estimation, demonstrating online navigation performance with live trajectory visualization. Deployed the algorithm on a TI C2000 microcontroller and experimentally validated the system across multiple in-campus vehicle trajectories, confirming improved localization accuracy under real-world conditions.

Intelligent multimodal drone and stealth aircraft detection and intent-based avoidance system

Detection of enemy drones and stealth aircraft

The proposed system is inspired by stealth hunting strategies of nocturnal predators such as owls, which use silent motion, environmental disturbance sensing, and predictive behavior to approach prey without detection. Similarly, the proposed UAV system detects environmental disruptions caused by stealth aircraft instead of relying only on direct sensing. The system integrates tri-modal sensing Atmospheric Lidar Tomography (ALT), Passive RF sensing, and Rotating Active RF Tomography (RART) with sensor fusion, intent prediction, and adaptive path planning to enable secure and intelligent drone navigation.

Publications & Patent

IEEE Sensors Letters

Novel GNSS-INS Integration Scheme With UKF-Based Dynamic IMU Calibration and Dual-layer Design for Reliable Navigation

In this work, a robust and improved loosely coupled (LC) Global Navigation Satellite System (GNSS)- Inertial Navigation System (INS) integration scheme incorporating three important features, such as dynamic inertial measurement unit (IMU) calibration, Mahalanobis distance-based outlier rejection (MDOR) mechanism, and innovation-based adaptive estimation (IAE) technique, is presented for reliable and accurate navigation. Unscented Kalman filter (UKF)-based dynamic calibration of IMU is adapted here because it accurately transmits statistical distributions without linearization, which is better at managing nonlinear INS dynamics than the extended Kalman filter. The navigation accuracy of the proposed method exhibits upto 75% improvement with respect to conventional LC integration.

IEEE INDICON Conference

Static and Dynamic Calibration of 9-Axis IMU Using UKF with Low Cost Setup

In this paper we have proposed a technique for both static and dynamic calibration of IMU sensor. We have developed a simple low-cost setup and used least square (LS) fit and ellipsoidal fit algorithm for static calibration and further incorporated Unscented Kalman filter (UKF) for dynamic calibration. The accelerometer is calibrated using 108 position data for three axis and gyroscope is calibrated using specially designed low-cost setup with LS algorithm whereas for magnetometer calibration, ellipsoidal fitting algorithm is used. Accordingly, all the calibration parameters like bias factor, scale factor and misalignment for accelerometer; bias factor, scale factor and non-orthogonality for gyroscope; Offset, gain and cross-gain for magnetometer are obtained. Moreover, for dynamic models like real-time or sensor fusion applications, the UKF (Unscented Kalman filter) based algorithm is proposed to obtain the calibration parameters dynamically.
